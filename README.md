# hikari-no-lounge-IR
ひかりのラウンジのIR(Impulse Response)データ

./wav/  
- hikari-IR(averaged+normalized).wav

## Usage

Ableton liveのHybrid Reverbなどで畳み込んでください。  
"hikari-IR(averaged+normalized).wav"をDrag & Dropするだけでok 👍

![abletom](./images/ableton-settings.png)
※Decayを小さめにするのがオススメ
